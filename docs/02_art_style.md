# 02 Art Style

## 总体方向

美术风格定位为“轻传奇高亮塔防肉鸽”。画面应保留传奇题材中沙城、城门、火墙、雷电、神兽、爆装等高识别元素，同时降低复杂写实度，让移动端小游戏用户能快速理解战场状态。

## 镜头与场景

- 采用竖屏 2.5D 斜俯视角。
- 核心场景为沙城守门：玩家阵线位于城门附近，怪潮从远端或下方路径压进。
- 场景应强调沙地、城墙、城门、烽火、破损旗帜、怪潮入口等元素。
- 战场空间需要为大字号伤害数字、爆装光柱、弱点靶心和三选一强化留出清晰层级。

## 角色方向

### 弓手

偏稳定单体输出，轮廓清晰，箭矢轨迹直接，适合作为基础 DPS 识别点。

### 火药师

偏范围爆破形象，投掷物和爆点需要明亮、集中、易读，适合形成直播围观爆点。

### 冰法师

偏减速控场形象，冰蓝色命中反馈要能让玩家看懂敌人被拖慢。

### 毒师

偏持续压制形象，可围绕毒雾、毒符、持续状态图标表现。

### 护卫

偏守门阻挡形象，轮廓厚重，站位稳定，强调拖慢近城敌人的安全感。

### 鼓手

偏节奏光环形象，鼓点、环形波纹和队伍增益反馈需要清楚但不遮挡战场。

### 治疗师

偏城池回复形象，治疗光效应温和、清晰，重点表达城池血量被拉回。

### 咒术师

偏破防增伤形象，可用暗色符文、破甲标识和易伤闪光表现。

## 特效方向

- 火墙：高亮、持续、成片压制，能一眼看出伤害区域。
- 雷电：瞬发、强闪光、清晰落点，适合表现暴击或高级强化。
- 毒符：轨迹清楚、命中有持续状态提示。
- 暴击数字：大字号、高对比、短促放大，优先保证可读。
- 靶心弱点：弱点标识应清晰贴合目标，命中时有明显破碎、闪光或音效反馈。
- 掉落光柱：纵向高亮，颜色区分品质，适合直播画面中被快速识别。

## UI 原则

面向中老年用户时，UI 应遵循：

- 大字号：核心数值、按钮文案、强化描述必须清楚。
- 大按钮：常用点击区域足够大，降低误触。
- 高对比：按钮、血条、奖励、警告状态要有明确色彩差异。
- 少层级：局内只保留必要信息，减少复杂菜单。
- 强反馈：点击、选择、升级、爆装、失败风险都要有清晰反馈。
- 直播友好：观众在小屏或压缩画质下仍能看懂当前爽点。
